용어
====

.. glossary::

   MWD
      Molecular Weight Distribution. chain length 또는 molecular weight 축으로 표현되는 polymer 분포.

   GPC
      Gel Permeation Chromatography. 분자량 분포를 실험적으로 관찰하는 대표 분석 방식.

   SEC
      Size Exclusion Chromatography. GPC와 유사하게 사용되는 분리/분석 용어.

   PBE
      Population Balance Equation. chain length 또는 particle size population의 동역학 방정식.

   Galerkin backend
      chain distribution을 basis coefficient로 표현하여 적분하는 backend.

   Recipe
      initial condition, feed, profile, integration control, schedule을 묶은 실행 조건.

   Generic Parameter
      ``GP_kp`` 처럼 reaction template이나 model equation에 binding되는 공통 parameter.

   Pre-schedule
      특정 시간 이후 feed rate, temperature, pressure, residence time 등을 바꾸는 recipe action.

   Shooting Control
      target output을 맞추기 위해 parameter를 반복 조정하는 계산 절차.
